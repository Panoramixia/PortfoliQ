<h1 align="center">PortfoliQ</h1>

## Project Description

This project goal is to help Quebec investors with their investments portfolio.

## Installation

1.  Click on the Green button Code
2.  Choose Download Zip
3.  Unzip the content in the folder of your choice
4.  Import the portfoliq.factory\_reset.json file to initialize the application
5.  Enter your information following the steps
6.  Do not forget to Export your information to a new json file when you are done or you will loose your progress
